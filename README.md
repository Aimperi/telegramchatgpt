# Telegram Recipe Bot 🍳

Telegram-бот для генерации рецептов на основе списка продуктов с использованием ChatGPT API.

## Описание

Бот помогает пользователям находить рецепты блюд на основе имеющихся у них продуктов. Просто отправьте список продуктов, и бот предложит два варианта рецептов:
- **Рецепт №1**: использует только указанные продукты
- **Рецепт №2**: может включать дополнительные ингредиенты

## Требования

- Python 3.11+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- OpenAI API Key (получить на [platform.openai.com](https://platform.openai.com))
- Railway аккаунт для развертывания

## ⚠️ ВАЖНО: Только Railway Deployment

**Локальная установка ЗАПРЕЩЕНА.** Проект разворачивается ТОЛЬКО на Railway Platform.

## Развертывание на Railway

### Шаг 1: Подготовка репозитория

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта в репозиторий
3. Убедитесь, что файлы `requirements.txt`, `Procfile` и `railway.json` присутствуют

### Шаг 2: Создание проекта на Railway

1. Перейдите на [railway.app](https://railway.app)
2. Войдите через GitHub
3. Нажмите "New Project"
4. Выберите "Deploy from GitHub repo"
5. Выберите ваш репозиторий

### Шаг 3: Настройка переменных окружения

В Railway Dashboard перейдите в раздел Variables и добавьте:

```
BOT_TOKEN=your_telegram_bot_token_here
OPENAI_API_KEY=your_openai_api_key_here
```

### Шаг 4: Деплой

Railway автоматически:
- Установит зависимости из `requirements.txt`
- Запустит бота через команду из `Procfile`
- Перезапустит бота при сбоях

## Использование бота

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Отправьте список продуктов (например: "курица, рис, морковь, лук")
4. Получите два рецепта с пошаговыми инструкциями

## Технологический стек

- **Python 3.11+**: Основной язык программирования
- **aiogram 3.x**: Асинхронная библиотека для Telegram Bot API
- **OpenAI API**: Генерация рецептов через ChatGPT
- **python-dotenv**: Управление переменными окружения
- **Railway**: Платформа для развертывания

- ✅ Форматирование ответов в Telegram Markdown
